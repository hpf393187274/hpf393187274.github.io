¶ 基础用法

123456789101112134950
1 / 50共 500 条

v-model 绑定之值 label 设置标签 placeholder 提示语句

<template>
  <me-paging :total="total"/>
</template>
<script>
export default {
  data() {
    return {
      total: 500
    }
  }
}
</script>
